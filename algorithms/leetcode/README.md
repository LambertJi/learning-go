# LeetCode 算法练习

Go 语言实现的力扣算法题

## 📁 目录结构

```
leetcode/
├── array/       # 数组类题目
├── string/      # 字符串类题目
├── linkedlist/  # 链表类题目
├── tree/        # 树类题目
├── dp/          # 动态规划
├── graph/       # 图论
├── sorting/     # 排序
├── search/      # 搜索
└── other/       # 其他类型
```

## 📝 文件命名规范

```
[题号]_[题目英文名]_简化版.go

示例：
0001_two_sum.go           # 1. 两数之和
0002_add_two_numbers.go   # 2. 两数相加
0003_longest_substring.go # 3. 无重复字符的最长子串
```

## 🎯 使用方法

### 1. 查看题目解法

直接打开对应的 `.go` 文件查看代码和注释

### 2. 运行测试

```bash
# 运行特定题目的测试
go test -v ./leetcode/array/0001_two_sum.go ./leetcode/array/0001_two_sum_test.go

# 运行某个类型的所有测试
go test -v ./leetcode/array/

# 运行所有算法测试
go test -v ./leetcode/...
```

### 3. 添加新题目

1. 在对应类型目录下创建文件
2. 复制模板代码
3. 实现解题函数
4. 编写测试用例
5. 运行测试验证
